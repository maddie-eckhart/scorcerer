# Scorcerer
A simple score-keeping app.

## Features
- Add/remove players
- Edit a player's score by increments of 1, 5, or 10
- Use a custom calulator to add/subtract from a player's score

## Future Features
- save players
- dark mode
- edit player
- filter players

### Feature Ideas
- player avatars
- use live activities to show current scores
- other tabs
  - decide first player
  - dice tracker
